 README
===================================================================================================================
Run the program from the command line interface with "java program2 [filename] [vehicle make] [vehicle model]", 
where [filename] is the desired CSV in the proper format.

	Example: c:\java program2 small_sample.csv Dacia Sundero
	
Output will be filtered based on the specified vehicle makes and models. The program will print sort times and the first 5/last 5 of each sorted list.
